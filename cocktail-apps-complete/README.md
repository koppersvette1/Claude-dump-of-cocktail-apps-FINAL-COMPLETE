## Version Overview

### v1 - Bar Companion AI (Your Production App)
**Status:** Deployed on GitHub at github.com/koppersvette1/v1-Bar-Companion-AI
**Tech Stack:** Full-stack (React 19, Node.js, PostgreSQL, Drizzle ORM)
**Key Features:**
- 1000+ built-in recipes with adaptive ranking
- Smoker Lab with 16+ wood profiles and guided timers
- Inventory management with barcode scanning
- Guest taste profiles with adaptive recommendations
- Pairing engine for food-to-drink suggestions
- Flights (side-by-side cocktail tasting planning)
- Cost tracking per drink
- Hybrid intelligence (deterministic rules + adaptive learning)

### v3 - MixMaster (Feature-Complete Rebuild)
**Status:** Built together in Claude conversations
**Tech Stack:** React with Tailwind CSS, TheCocktailDB API
**Key Features:**
- TheCocktailDB integration (636+ cocktails)
- "What Can I Make?" ingredient matching
- Advanced search and filtering
- My Bar inventory tracking
- Favorites system
- Mobile-first bottom navigation

### v4 - MixMaster Enhanced (UI Polish + v1 Features)
**Status:** Built to match v1's elegant design
**Tech Stack:** React with shadcn/ui patterns
**Key Features:**
- All v3 features PLUS:
- Smoker Lab with wood profiles
- People profiles for personalization
- Card-based elegant UI
- Detailed cocktail information (technique, difficulty, ABV, pro tips)
- Enhanced information hierarchy
