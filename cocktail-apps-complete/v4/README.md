# MixMaster v4 - Enhanced Version

Polished version with Smoker Lab, People profiles, and elegant shadcn/ui-inspired design.

## New in v4
- Smoker Lab with wood profiles
- People profiles for taste personalization
- Card-based UI layouts
- Detailed cocktail information (technique, difficulty, ABV, tips)
- Enhanced visual hierarchy

## All v3 Features Plus
Everything from v3 PLUS the enhancements above.

## Running
```bash
npm install
npm start
```

## Tech Stack
- React 18 with Hooks
- shadcn/ui design patterns
- Mobile-first responsive design
